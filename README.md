# L4D2-NPC-VScript-
The L4D2 VScript mod that I made that adds Dummy NPCs around the map and for my upcoming L4D2 Mod

I borrowed some voice lines assets from Bully and POSTAL 2 video games, their sounds belong to them.

This mod basically in short spawn prop_dynamic npc with brains, it currently have Penguin model as a placeholder
The NPC have 4 classes: Non-armed (normal), Armed, Projectile Armed and Melee armed, they also react to survivors and infected.

You are can edit this script and use it but please give me a credit!
